# 形式语言与自动机大作业

本项目实现了正则表达式解析的全流程，包括从正则表达式到最小化 DFA 和正则文法的转换，并实现了去陷阱态和正则匹配的功能。


---

## 功能简介

本程序支持以下功能：
- 正则表达式解析：RE -> ε-NFA -> NFA -> DFA（含陷阱态） -> 最小化 DFA -> RG（正则文法）。
- 另外添加了去除陷阱态的功能。
- 添加了正则匹配功能，可以输入正则表达式和字符串，判断字符串是否符合正则表达式。
- 支持的运算符：
  - **连接**：`ab`
  - **或**：`a+b`
  - **闭包**：`a*`
  - **括号**：`(a+b)*`
- 支持的输入符号：
  - **字符**：`0`, `1`
  - **括号**：`()`
  - **运算符**：`+`, `*`
- 输出：
  - 最小化 DFA
  - 正则文法
